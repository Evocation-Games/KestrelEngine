
# Property: graphicsLayerName
### `Kestrel.graphicsLayerName`

| Aspect | Value |
| --- | --- |
| Type | `int` |
| C++ Getter Function Symbol | `kestrel::renderer_api_name()` |
| Introduced | 0.8.0 |

#### Getter Description
Returns the name of the current graphics API being used by Kestrel.
- `None`
- `OpenGL`
- `Metal`
