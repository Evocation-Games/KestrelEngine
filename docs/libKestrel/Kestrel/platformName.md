
# Property: platformName
### `Kestrel.platformName`

| Aspect | Value |
| --- | --- |
| Type | `int` |
| C++ Getter Function Symbol | `kestrel::platform_family_name()` |
| Introduced | 0.8.0 |

#### Getter Description
Returns the name of the current platform that Kestrel is running on.
- `Unknown`
- `macOS`
- `Windows`
- `Linux`
