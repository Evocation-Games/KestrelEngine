Angle

cos(magnitude)

Aspect Value
Source File libKestrel/math/angle.hpp
C++ Symbol kestrel::math::cos
Introduced Version 0.8.0

Warning

This method is more accurate than `Angle:fcos()`, but it is slower. Make sure you use the most appropriate version for your use-case.

Description

Compute the cosine of the angle to the specified magnitude.

Parameters

Parameter Type Description
magnitude ? The value that the cosine will be multiplied by.
