Source File libKestrel/kestrel.hpp
C++ Symbol kestrel::platform_family
Introduced Version 0.8.0

Description

Provides a value representing the current platform that Kestrel is running on. - `Platform.Family.Unknown` - `Platform.Family.MacOS` - `Platform.Family.Windows` - `Platform.Family.Linux`

Return Value

Platform.Family.
